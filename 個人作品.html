AutoUpdater

A little program to auto update your program.

Github網址：

https://github.com/flier268/AutoUpdater

使用方法：

將AutoUpdater.exe、AutoUpdater.ini與你的程式放置於同一目錄下 需要檢查更新時執行AutoUpdater.exe 可參考此程式碼：
  1. void Update()    
  2.  {    
  3.      ProcessStartInfo startInfo = new ProcessStartInfo("AutoUpdater.exe");    
  4.      startInfo.WindowStyle = ProcessWindowStyle.Minimized;    
  5.      startInfo.Arguments = "-CheckUpdateWithForm";    
  6.      Process.Start(startInfo);    
  7.  }    

有三種啟動參數：
-updateself" 檢查AutoUpdater是否有更新
-CheckUpdateWithForm 按照AutoUpdater.ini的設定更新，會顯示Form
-CheckUpdateWithoutForm 按照AutoUpdater.ini的設定更新，不會顯示Form

只支援ZIP的壓縮格式，其他的不支援(7z、rar...)

配置文件：

<AutoUpdater.ini>

;這個檔案與AutoUpdater.exe放於同一目錄下
  1. [AutoUpdater_URL]    
  2. APP_Info_ini=https://sites.google.com/site/pow7000web/mysoftware/autoupdater/範例程式.ini    
  3.   
  4. [APP_Info]    
  5. APP_Name=範例程式    
網址那裏必須是一個可以下載的ini檔

<範例程式.ini>
